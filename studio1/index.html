Code Style Story

Every Friday we would have labs where we would work in groups of two. My partner and I had a lot of fun making our programs look odd and confusing.

We would call variable names “eye” instead of “i” or “ex” instead of “x” and put them in for-loops like

for(eye; eye < 100; eye++)

Sometimes we would put lots of stuff on one line with no new lines.

We would also write many silly comments about obvious code. Our professor definitely did not like it… He-he-he!!!
